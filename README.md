# Dumpster Dudes Website

Trash can valet service for Paso Robles, CA (93446).

## About

Dumpster Dudes is a residential trash can valet service started by Luke K., a high school cross country runner, to fund his running dreams. We take your bins to the curb and bring them back - rain or shine, every week, for just $25/month.

## Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Hosting:** [Netlify](https://netlify.com/)
- **Forms:** Netlify Forms

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site auto-deploys to Netlify on push to the `main` branch.
